# Lume Android App

This area owns Android-specific packaging and shell behavior.

## Current Runtime

- Shared Rust core: `crates/lume_engine`
- Config overlay: `src-tauri/tauri.android.conf.json`
- Native project: generated by `npm run tauri:android:init`.
- Artifact output folder: `apps/android/installers`.

## Build

```powershell
npm run build:android
```
