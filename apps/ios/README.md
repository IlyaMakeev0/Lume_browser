# Lume iOS App

This area owns iOS-specific packaging and shell behavior.

## Current Runtime

- Shared Rust core: `crates/lume_engine`
- Config overlay: `src-tauri/tauri.ios.conf.json`
- Native project: generated by `npm run tauri:ios:init`.
- Artifact output folder: `apps/ios/installers`.

## Deferred Build

```bash
npm run build:ios
```

iOS packaging requires macOS, Xcode, signing, and provisioning.
