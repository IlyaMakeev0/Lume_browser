# Lume

Lume is a Tauri 2 desktop shell with a Rust backend and a Next.js + Tailwind CSS frontend.

## Project Parts

Lume is split into five product areas:

- `apps/windows` - Windows distribution and WebView2 shell notes.
- `apps/macos` - macOS distribution and WKWebView shell notes.
- `apps/android` - Android mobile shell notes for Tauri-generated native code.
- `apps/ios` - iOS mobile shell notes for Tauri-generated native code.
- `crates/lume_engine` - shared Rust browser engine core used by every shell.

The active Tauri shell lives in `src-tauri` and calls into `crates/lume_engine`.

## Development

Install Node.js 20.9+ and Rust 1.85+ through rustup, then run:

```bash
npm install
npm run tauri:dev
```

The frontend is configured for static export into `dist/`, which is consumed by Tauri.

## Packaging

Windows and Android build scripts are prepared:

```powershell
npm run build:windows:x64
npm run build:windows:arm64
npm run build:android
```

Packaging outputs are documented in `PACKAGING.md`.

Mobile projects are generated by Tauri when the native SDKs are installed:

```bash
npm run tauri:android:init
npm run tauri:ios:init
```
